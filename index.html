ABOUT THE EVENT

CIREBON QRIS RUN 2026 hadir sebagai event lari modern dengan konsep ultra premium, energetic, dan cinematic experience — BUKAN SEKEDAR FUN RUN! Dirancang untuk menghadirkan lebih dari sekadar perlombaan lari.

Diselenggarakan di Grage Mall, Cirebon pada Sabtu, 16 Agustus 2026 sebagai bagian dari TRIKARSA FEST, event ini menggabungkan semangat olahraga, kebersamaan, gaya hidup sehat, dan aksi sosial dalam satu pengalaman spektakuler.

Dengan kategori 5K Run dan 10K Run, CIREBON QRIS RUN 2026 terbuka untuk seluruh kalangan. Jelajahi ikon Kota Cirebon, dukung UMKM lokal, dan rasakan kemudahan transaksi digital dengan QRIS!
